Запуск:
Чтобы запустить калькулятор, нужно открыть файл index.html
Функции:
sqrt - квадратный корень от числа. Ввод "Число" -> "sqrt" -> "="
ce - очистка второй переменной и знака операции
ac - полная очистка
del - удаление последнего символа
x^y - возведение первой переменной в степень со значением второй
+/- изменение знака переменной
% - вычисление процента от числа. Ввод "Число" -> "%" -> искомый процент от числа -> "="
/ - деление
X - умножение
- - вычитание
+ - сложение
= - вывод результата
